# Ecommerce Data Analysis Project

## Project Description
This project analyzes sales data from an online store to identify trends, the most profitable products, and growth opportunities. It demonstrates how a data analyst can use historical transaction and customer data to guide business decisions.

## Dataset
The dataset used is `synthetic_ecommerce_data.csv`, which contains information about transactions, customers, products, and marketing campaigns.

## Objectives
- Analyze sales trends by category and region.
- Calculate key KPIs such as revenue per unit, conversion rate, and ROI per campaign.
- Identify the most profitable products and the most valuable customers.

## Key Findings / Conclusions
- Electronics and Home Appliances are the categories with the highest revenue.
- Asia shows the best ROI from marketing campaigns.
- Campaigns with higher CTR tend to generate more conversions.

## How to View
- Open `Ecommerce_Data_Analysis.html` in your web browser to see the full analysis, including graphs and tables.
- The `synthetic_ecommerce_data.csv` file contains all the necessary data for reproduction.
